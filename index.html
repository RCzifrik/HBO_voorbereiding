Welkom op onze applicatie.

Op onze website vind je een overzicht van ons project. In ons project moesten MBO studenten informatie geven over het overstappen naar het HBO. Op onze website vind je onze onderzoeksvragen die we hebben voorgesteld aan HBO docenten en leerlingen. Hiermee hopen wij jou meer informatie te kunnen geven.

Op onze website kun je de doorstroommogelijkheden vinden en de vaardigheden die je nodig hebt. Ook geven we je tips zodat je je makkelijker kan voorbereiden op de overstap.

We hebben onderzocht wat nodig is om de overstap te maken doormiddel van de juiste informatie en voorbereiding zal de overstap een stuk makkelijker worden.

Website medemogelijk gemaakt door de leerlingen van ROC Mondriaan.
